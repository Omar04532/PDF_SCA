# 📚 KDP Children's Books Research Suite Pro

مجموعة أدوات مجانية متكاملة مبنية على **Streamlit** لناشري كتب الأطفال على Amazon KDP — النسخة المحسّنة.

## ✨ ما الجديد في النسخة Pro؟

| الميزة | الوصف |
|--------|-------|
| **🔌 زر تشغيل/إيقاف** | يمكنك تعطيل الأداة واستخدام بيانات سابقة |
| **🛡️ الوضع الآمن** | تأخير ديناميكي بين الطلبات لتجنب حظر أمازون |
| **💾 ذاكرة مؤقتة (Cache)** | تخزين نتائج البحث لمدة تصل إلى 168 ساعة |
| **🌐 مدير البروكسيات** | تدوير IP تلقائي عند الحظر + جلب بروكسيات مجانية |
| **🛑 إيقاف طارئ** | زر إيقاف البحث فوراً أثناء التنفيذ |
| **📊 تصنيف ذكي** | تصنيف الكلمات حسب النيتش ونية البحث والمنافسة |

## 🚀 التشغيل المحلي

```bash
# 1. استنساخ المستودع
git clone https://github.com/YOUR_USERNAME/kdp_research_suite.git
cd kdp_research_suite

# 2. إنشاء بيئة افتراضية
python -m venv venv

# 3. تفعيل البيئة
# Linux/Mac:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# 4. تثبيت المتطلبات
pip install -r requirements.txt

# 5. تشغيل التطبيق
streamlit run app.py
```

## 🌐 النشر على Streamlit Cloud (مجاني)

1. ارفع المستودع على GitHub
2. اذهب إلى [share.streamlit.io](https://share.streamlit.io)
3. اربط حساب GitHub واختر المستودع
4. اضغط **Deploy** — التطبيق يعمل!

## 📁 هيكل المشروع

```
kdp_research_suite/
├── app.py                    # التطبيق الرئيسي (النسخة المحسّنة)
├── requirements.txt          # المكتبات المطلوبة
├── README.md                 # دليل المشروع
├── .gitignore               # ملف استبعاد
├── cache/                    # مجلد الذاكرة المؤقتة (يُنشأ تلقائياً)
└── utils/
    ├── __init__.py           # تنظيم الاستيراد
    ├── keyword_tools.py     # أدوات تصنيف الكلمات
    ├── proxy_manager.py     # إدارة البروكسيات الدوارة
    └── cache_manager.py     # إدارة الذاكرة المؤقتة
```

## ⚠️ تنبيهات مهمة

1. **الذاكرة المؤقتة:** مجلد `cache/` يُنشأ تلقائياً. لا ترفعه على GitHub إذا كان يحتوي على بيانات حساسة (أضفه في `.gitignore`).
2. **البروكسيات:** البروكسيات المجانية قد تكون بطيئة. للاستخدام المكثف، أنصح بـ [ZenRows](https://www.zenrows.com/) أو [ScrapingBee](https://www.scrapingbee.com/) (لديهما باقات مجانية).
3. **حظر أمازون:** إذا واجهت حظراً متكرراً، فعّل "الوضع الآمن" وزِد التأخير إلى 2-3 ثوانٍ.
